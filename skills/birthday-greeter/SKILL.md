# Birthday Greeter

## Purpose

Generates a heartfelt, personalised birthday message for a family member by
drawing on memories stored in the memories/ folder.

## How to execute this skill

1. Ask who the birthday message is for and who it is from.

2. Read all files inside the memories/ folder.

3. Find any memories that mention or involve the birthday person.

4. Write a birthday message that:
   - Opens with a warm, personal greeting
   - References at least one specific real memory from the memories/ folder
   - Expresses what that person means to the family
   - Closes with a loving birthday wish

5. Show the message to the user and ask:
   "Does this feel right? Would you like me to adjust the tone or add anything?"

6. Revise if needed, then present the final version.

## Tone guidelines

- Warm, genuine, never generic
- Reads like it was written by someone who truly knows the person
- Avoid clichés like "wishing you all the best" — be specific and personal
- Length: 3–5 short paragraphs
